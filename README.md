# Aapki Apni Dukan

A responsive e-commerce website made using HTML, CSS, and JavaScript. It fetches product data from an API and includes features like search, add to cart, quantity management, and saving the cart using localStorage.

## Features

- Product listing from API  
- Search products  
- Add to Cart  
- Quantity management  
- Remove items from cart  
- Cart saved using localStorage  

## Tech Used

- HTML  
- CSS  
- JavaScript  

## How to Run

1. Download the project  
2. Open index.html in browser
